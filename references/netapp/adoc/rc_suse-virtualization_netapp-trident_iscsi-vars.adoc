:subtitle: A Step-by-Step Guide to Configuring Persistent Storage for Virtual Machines Using NetApp ONTAP and Trident CSI
:usecase: Persistent Storage for VMs using iSCSI
:description: This guide details the steps to configure NetApp external storage with a SUSE Virtualization cluster using Trident CSI for VM persistent volumes.
